# Release 1.0.0

本目录包含与源码版本 1.0.0 配套的可烧录文件。

- `MCU/release_yuexinmiao.hex`：NUC131 APROM 固件。
- `MCU/release_yuexinmiao.bin`：MCU 原始 BIN。
- `MCU/build_log.txt`：Keil 强制全量编译日志。
- `GD25Q64/release_yuexinmiao_gd25q64_8MiB.bin`：推荐整片烧录镜像。
- `GD25Q64/release_yuexinmiao_animation_pack.bin`：实际长度资源包。
- `GD25Q64/animation_manifest.json`：资源总清单和逐帧数据。
- `GD25Q64/animation_frames.csv`：逐帧人工核对表。
- `GD25Q64/validation_report.json`：离线解码和镜像校验结果。

原始猫咪 MP4 和预览 MP4 不包含在开源包中。
